# 🍳 Web Essen - Panel de Administración

Sitio web oficial para Essen Argentina con sistema de administración de productos.

## ✨ Características Principales

- 🎨 **Diseño moderno y responsive** - Se adapta a celulares, tablets y computadoras
- ⚡ **Panel de administración completo** - Gestiona productos fácilmente
- 🛒 **Sistema CRUD completo** - Crear, Leer, Actualizar y Eliminar productos
- 💾 **Almacenamiento en JSON** - Los datos se guardan permanentemente
- 🌐 **API RESTful** - Interfaz para futuras integraciones
